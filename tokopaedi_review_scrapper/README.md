- ⚡ Export data quickly for reporting and research

## 📦 Requirements

- `Python 3.11+`
- `bs4`
- `pandas`
- `PyYAML`
- `rich`
- `selenium`

---

## ⚙️ Installation

Install `uv`:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Sync dependencies:

```bash
uv sync
```

> This installs all dependencies from `uv.lock`.

---

## 🚀 Usage

Run the application:

```bash
cd password_manager
uv run app.py
```
